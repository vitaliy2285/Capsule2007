Capsule2007 — СИЛЬНО / Netlify project structure

Структура:
- index.html
- css/style.css
- js/app.js
- assets/images/ — вынесенные изображения
- assets/audio/ — фоновая музыка, ICQ-звук, звук открытия/закрытия ячейки

Как залить на Netlify:
1. Распакуй ZIP.
2. Перетащи папку Capsule2007_SILNO_NETLIFY_PROJECT в Netlify Deploys / Drop.
3. Открывай сайт по ссылке.

Важно:
- Фоновая музыка запускается после первого клика/тапа по сайту — это ограничение браузеров.
- Это PROD-структура: отдельные файлы удобнее править и быстрее грузить.
